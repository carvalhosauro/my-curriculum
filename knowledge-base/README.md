# Base de conhecimento (Agent-Like)

Esta pasta existe para você registrar seus feitos uma vez e reutilizar esse histórico para:

- atualizar curriculos com consistencia;
- adaptar versoes para cada vaga;
- gerar relatorios por contexto (vaga ou atribuicao).

## Estrutura

- `feitos.md`: base principal de feitos (fonte unica da verdade).
- `vagas/`: contexto e requisitos de cada vaga.
- `atribuicoes/`: foco por tipo de atribuicao/responsabilidade.
- `relatorios/`: saidas padronizadas para revisão e exportacao.
- Skill de automacao: `.cursor/skills/curriculo-agent-like/SKILL.md`.

## Fluxo rapido

1. Registre um novo feito em `feitos.md`.
2. Atualize/adicione um contexto em `vagas/` ou `atribuicoes/`.
3. Gere um relatorio a partir dos templates da pasta `relatorios/`.
4. Use o relatorio para ajustar um curriculo em `v0/`.

## Regra de ouro

Sempre escreva feitos com:

- contexto;
- acao;
- tecnologia;
- impacto mensuravel;
- evidencias (se houver).
